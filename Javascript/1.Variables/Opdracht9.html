Opdracht 9: Voor deze opdracht is het handig om je in te lezen over increment en decrement. 1. Copy de volgende code in jouw VSCode: ​let​ ​number​ = ​3​; ​number​++; 2. Print uitkomst in jouw browser console. Wat gebeurt er met de uitkomst? De uitkomst is 4. 3. Wijzig number++ naar number--. 4. Wat is de uitkomst van de som ? De uitkomst is 2. Wat gebeurt is de uitkomst? 5. Copy de volgende code in jouw VSCode: ​let​ ​number​ = ​3​; ​const​ ​sum​ = ​number​++ + ​2​; ​console​.​log​(​sum​); 6. Wat is de uitkomst van van de som ? Leg jouw antwoord uit. De uitkomst van de som is 5 omdat je eerst de waarde geeft en daar 2 erbij optelt. Pas na de uitkomst wordt er dus 1 opgeteld bij het getal met een waarde van 3. 7. Copy de volgende code in jouw VSCode: ​let​ ​number​ = ​3​; ​const​ ​sum​ = ++​number​ + ​2​; ​console​.​log​(​sum​); 8. Wat is de uitkomst van van de som ? De uitkomst van de som is 6 omdat je eerst 1 optelt bij number met een waarde van 3 en daarna pas 2 erbij optelt.
