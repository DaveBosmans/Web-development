Opdracht 5: Maak nu een Person object door gebruik te maken van Object initializers. Wat zou eventueel de voor en nadelen t.o.v Object constructors?

Als je kleine objecten gerbuikt kan je beter Object initializers gebruiken omdat het compacter is. Het pakt alles en propt alles in één lijn. Nadelen: het is heel onhandig met grote objecten die je vaker wilt gebruiken.
