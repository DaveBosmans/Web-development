Deze applicatie moet het volgende kunnen doen: ● Het vertalen van getallen naar maanden. Bijvoorbeeld, als je het getal 1 invoert dan toon je op het scherm “Januari”. ○ 1 => January. ○ 2 => February. ○ 3 => March. ○ 4 => April. ○ 5 => May. ○ 6 => June. ○ 7 => Juli. ○ 8 => August. ○ 9 => September. ○ 10 => Oktober. ○ 11 => November. ○ 12 => December.
