Opdracht 3 Er zijn verschillende selectors, bijvoorbeeld; type selectors descendant selectors class selectors Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

type selectors: De CSS-type selector komt overeen met elementen op knooppuntnaam. Met andere woorden, het selecteert alle elementen van het gegeven type binnen een document. Voorbeeld: a color: red, hiermee selecteer je alle anchors op de pagina. descendant selectors: De descendant selector komt overeen met alle elementen die afstammelingen zijn van een opgegeven element. voorbeeld: ul li selecteren alle list items in een ul. class selectors: De CLASS-selector heeft als basis een punt gevolgd door een class-naam. Hiermee selecteer je de elementen die die class name hebben. Voorbeeld: .my-class.
