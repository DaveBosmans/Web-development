Opdracht 3 Wat gebeurt er als je een width en een height toevoegt aan de span tag ( <span> ) van vorige opdracht?

Er gebeurd niks want inline elementen hebben geen eigen height en width tenzij je er een block element van maakt.

Dit is een block box en neemt de hele breedte in waardoor de volgende box op de volgende regel zit

In dit blok zit inline tekst in wat dus niet op de volgende regel komt te staan
