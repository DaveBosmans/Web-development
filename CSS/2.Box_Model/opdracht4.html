Opdracht 4 Hieronder zie je een box-model. Leg uit wat de volgende termen betekenen: content, padding, margin en border.

Content: Houdt in dat wat je content is. Dus alles in binnenin je element.

Padding: padding is de ruimte die je element in beslag neemt binnenin je element.

Margin: is de ruimte die om je element heen zit.

Border: is de ruimte tussen de padding en de margin in.
