Opdracht 1 Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Static

Dit is de standaard postion waarde van elementen in html. Een element is niet gepositioneerd op een speciale manier. Het element is gepositioneerd gebaseerd op het normale verloop van het document.

Relative

Position relative zorgt ervoor dat het element gepositioneerd is relatie van zijn normale postitie. Andere elementen zullen niet verplaats worden als er een lege ruimte ontstaat door dit element.

Absolute

Als je element absolute is dan verdwijnd deze van de document flow, dit element neemt dan geen ruimte meer in op de pagina. Zijn positie is relatief van de positie van zijn dichtbijzijnste ouder.

Fixed

Het element wordt verwijderd van de normale document flow en er is geen ruimte gecreeerd voor de element op de pagina layout. Positie van de element is nu relatief van de viewport.

Sticky

Met sticky blijft het element boven op beeld in dezelfde positie, als je naar beneden scrolled.
