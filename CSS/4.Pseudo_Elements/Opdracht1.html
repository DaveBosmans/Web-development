Opdracht 1 Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Een pseudo-element kun je zien als een denkbeeldig element, dat weliswaar niet in het document voorkomt, maar waarvoor je wel een stijl kunt definiÃ«ren. Je kant geen psuedo elementen plaatsen op elementen die geen content hebben bijvoorbeeld een input. Een psuedo element vraagt om content.

4 voorbeelden:

1. ::before: met pseudo element before komt de content van dit element VOOR de CONTENT van het element dus niet voor de element maar in de content.

2. ::after: met pseudo element after komt de content van dit element NA de CONTENT van het element.

3. ::first-letter : first letter zorgt ervoor dat elke de eerste letter van een tekst kan worden aangepast in style. Dit element kan alleen worden toegevoegd aan block level elementen.

4. ::first-line : Dit element wordt gebruikt om een speciale style toe tevoegen aan de eerste lijn van een tekst.
