Opdracht 2 Er zijn verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere ook benoemen en aangeven wat ze precies doen?

1. ::selection : Stel iemand selecteert een stukje tekst op je site, dan kun je met het pseudo element "selection" bepalen hoe de styling van dit geselecteerde stukje tekst moet zijn.

2. :: placeholder : Met placeholder kan je een tijdelijke aanduiding geven in een formulier. Bijvoorbeeld bij email: kan je email: example@hotmail.com invoeren als placeholder zodat mensen weten dat ze daar hun email moeten invullen.

3. ::backdrop : is een box dat het full screen element zoals een video achter zich heeft, maar tegelijkertijd zit het bovenop alle andere content van het element. Het kans gebruikt worden met de ::fullscreen class zo kan je de achtergrondkleur veranderen van de video in fullscreen.
