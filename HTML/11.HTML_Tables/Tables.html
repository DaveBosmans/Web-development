10.Om een tabel te maken heb je volgende elementen nodig: ● table ● thead ● tbody ● tfoot ● th ● tr ● td Geef voor elke element een definitie

● table : table tag defineert een HTML table.
● thead : thead gebruiken we om headers te groeperen.
● tbody : tbody gebruiken we om alle basis content te groeperen.
● tfoot : tfoot gebruiken we om voor content die we als footer zien.
● th : Dit zijn de table headers.
● tr : tr zijn de rijen in de table.
● td : td tags zijn de tabelcellen waar dus de content in komt.

13. Voor het maken van een tabel kan je ook gebruik maken van colspan en rowspan attributen. Zoek uit wat colspan en rowspan attribuut doet en meet het volgende:

What does colspan= do? Allows a single table cell to span the width of more than one cell or column. What does rowspan= do? Allows a single table cell to span the height of more than one cell or row.
