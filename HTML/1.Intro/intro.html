2. Wat is het verschil tussen HTML, CSS en JavaScript?

De grootste verschil tussen HTML en CSS is dat het HTML is een opmaaktaal die wordt gebruikt om de structuur van een webpagina te maken, terwijl CSS een stijltaal is die wordt gebruikt om de webpagina's beter te presenteren. 3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.

Een tag is een opdracht die door een browser kan worden geïnterpreteerd. HTML gebruikt tags om bijvoorbeeld de positie en weergave van elementen op een pagina aan te geven.
Een HTML-element is een onderdeel van het Document Object Model dat wordt gebruikt om webpagina's met HTML gestructureerd weer te geven. Een element kan meestal een of meer kenmerken hebben die een naam en een waarde bevatten.
Een tag staat altijd tussen een gesloten haakje en een openings haakje. Het betreffende element wordt omsloten door een begintag en een eindtag: element

7. Wat is het verschil tussen inline en block elementen?

Een block-level element begint altijd op een nieuwe regel en neemt de volle breedte van het browservenster in. Het strekt zich uit naar links en rechts uit zo ver als het kan. Een inline element start niet op een nieuwe regel en beslaat slechts zo veel breedte als het nodig heeft om wat erin staat weer te geven.
kenmerken van inline elementen: respecteer de linker- en rechtermarges en opvulling, maar niet boven en onder; kan geen breedte en hoogte hebben; laat andere elementen links en rechts van hen.
kenmerken van block elementen: respecteer alle margins and padding; een regelafbreking forceren na het blokelement.
